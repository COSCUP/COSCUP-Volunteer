# 文件撰寫

在籌備期間的各工作任務最後希望可以整理成一份文件紀錄，對於下次或是需要改善流程時能有參考的、實際的工作流程。

## 要領

撰寫工作任務紀錄時希望包含：

- [x] 工作任務的目的
- [x] 沿革（他曾經是什麼、現在變成什麼）
- [x] 任務的目標對象
- [x] 發生的時間點或是相對的時間
- [x] 任務執行的過程紀錄
- [x] 跨組或與廠商協調的項目
- [x] 檢討紀錄（下次可以換個方式、這次有哪些項目沒完成或原因）
